--------------------------------------------------------

## Iniciando um pacote com npm 

```
    npm init
```
ou
 iniciando dizendo sim para todas perguntas
```
    npm init -y 
```

## instalando dependências globais

 
 npm i -g para instalar global
```
    npm i opn -g
```
 
 npm root -g para encontrar local onde foi instalado
```
    npm root -g
```

## Desinstalar pacotes

```
    npm uninstall opn -g
```
