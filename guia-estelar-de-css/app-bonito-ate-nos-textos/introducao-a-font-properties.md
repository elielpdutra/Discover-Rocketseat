## Basic Font Properties

* font-family
* font-weight
* font-style
* font-size

--------------------------------------

## Font Family

* Tipo de fonte de um elemento
* Lista de fontes e ordem de prioridade
* Inclui *fallback* font

```css
p {
    font-family: "Times New Romes", Times, serif;
}
```

    - serif
    - sans

----------------------------------------

## Font Weight

* Peso da fonte

```css
span {
    font-weight: bold;
}
```

----------------------------------------

## Font Style

* O Estilo da fonte

```css
span {
    font-style: italic;
}
```

----------------------------------------

## Font Size

* O tamanho da fonte

```css
p {
    font-size: 10px;
}
```
